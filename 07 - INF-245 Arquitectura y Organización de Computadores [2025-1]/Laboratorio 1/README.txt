Explicación:
Usamos clases para representar a las entidades (Snake, guardia, objetivo).
El mapa es una matriz que posiciona aleatoriamente a las entidades (según las reglas del enunciado)
La conversión de bases se hace con una sola función que permite convertir a los 3 tipos.
Para transformar decimal a binario, usa el método de divisiones sucesivas entre 2.
Para decimal a octal es idéntico, pero divide entre 8.
Y para hexadecimal, divide entre 16 y mapea los caracteres a un string con los caracteres soportados.

Supuestos:
El usuario ingresará datos válidos por consola.
El usuario no ingresará valores altos para el movimiento.
Un pasillo chico es hasta tamaño 10, mediano es hasta 50 y grande es mayor a 50.
Al chocar contra una pared, se aparecerá en la posición opuesta.
