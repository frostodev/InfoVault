Nosotros usamos Microsoft SQL Server 2022 Express como nuestro servidor de base de datos, y SQL Server Management Studio en Windows 10.

Primero se crea una base de datos vacía.
Luego, se deben crear las tablas usando el archivo "tablas.sql"
Después, se importan los datos desde los archivos csv usando "insert.sql"
Nótese que "insert.sql" tiene la ruta "D:\athlete_events.csv" y "D:\noc_regions.csv", por lo que debe ser cambiada acorde a donde están los archivos.
Finalmente, las consultas se encuentran en el archivo "consultas.sql", separadas por comentarios.
