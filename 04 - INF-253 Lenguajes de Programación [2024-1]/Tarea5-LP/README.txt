Se probó en Windows 10 y Windows 11; con SWI-Prolog 9.2.5
Para cargar cada programa, se debe abrir SWI-Prolog, ir al menú File y elegir la opción "Consult ..."

Desde ahí, se debe elegir uno de los programas.
Para probarlo, se debe escribir en la terminal de SWI-Prolog el predicado con sus correspondientes argumentos.
Ejemplo:

?- obtenerRango([5, 2, 3, 1, 9], R).
